# Breast Cancer Prediction App

A machine learning web application that predicts whether a tumor is malignant or benign based on input features.

## 🔗 Live Demo
https://breast-cancer-predictor-kanishka.streamlit.app

## 🚀 Features
- Interactive UI using Streamlit
- Real-time prediction based on user inputs
- Pretrained ML model with feature scaling
- End-to-end pipeline (data → model → deployment)

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📂 Project Structure
- app.py – Streamlit application
- model.pkl – trained ML model
- scaler.pkl – feature scaler
- requirements.txt – dependencies

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py

## 📌 Notes
Pretrained model is included for demonstration purposes.
